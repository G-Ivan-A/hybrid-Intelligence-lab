# NIST AI RMF Profile Loop

## Source, author, link

- Source: NIST AI Risk Management Framework 1.0 and AI RMF Playbook.
- Organization: National Institute of Standards and Technology.
- Link: https://www.nist.gov/itl/ai-risk-management-framework

## Practice

Use a lightweight Map, Measure, Manage, Govern loop for AI-assisted work in Hub
projects.

## Problem

AI risk work often stays abstract unless each project maps context, measures
specific risks, manages mitigations, and assigns governance ownership.

## Apply In Hub

1. Map: define the AI-assisted workflow, users, data, tools, and impacted
   artifacts.
2. Measure: identify quality, privacy, safety, and misuse risks that can be
   tested or reviewed.
3. Manage: choose mitigations such as human review, scope limits, logging,
   validation scripts, and rollback.
4. Govern: assign owner, review cadence, and escalation path in the issue or
   project governance file.

## Output

A project profile or issue checklist that names risks, mitigations, owners, and
evidence.

## Limits

Use the loop proportionally. A draft prompt does not need the same weight as a
public AI-facing feature.
